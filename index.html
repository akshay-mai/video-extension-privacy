Privacy Policy

Last updated: March 2026

This privacy policy describes how the Video Player Controller browser extension handles information when users install and use the extension.

Overview

The extension allows users to control video playback on supported streaming websites such as Prime Video, Hotstar, and Crunchyroll. Features may include play, pause, and picture-in-picture controls for video players.

Data Collection

This extension does not collect, store, or transmit any personal information.

The extension only interacts with video elements present on the webpage to enable playback controls.

Permissions Usage

The extension requires the following permissions:

  • activeTab – Used to access the currently active tab when the user activates the extension.
  • scripting – Used to inject a script that detects and controls video elements on supported webpages.
  • Host Permissions – Required to allow the extension to function on supported video streaming websites.

Data Sharing

This extension does not collect user data and does not share any information with third parties.

Remote Code

The extension does not load or execute remote code. All functionality is contained within the extension package.

Changes to This Policy

If this privacy policy changes in the future, updates will be reflected on this page.

Contact

If you have any questions regarding this privacy policy, you may contact the extension developer.
